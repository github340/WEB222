• Explain the concept of Responsive Design. What does the term mean?

A Responsive Design is a technique that automatically changes the size of a screen depending on the type of display on a website.

• How does responsive design relate to web technologies?

A Responsive Design is a web development approach that dynamically changes the appearance of a website depending on the screen size and orientation of the device used to view it. A Reponsive Design rely on proportion-based grids to rearrange content and design elements.

• What did you learn about building web apps for different devices through your research?

The first thing to consider in building a responsive website is layout. When building a responsive website, it always begins by creating a non-responsive layout that is fixed to the basic size. If developers have created a non-responsive layout, create a responsive site by adding some changes to the media query and CSS. Media query consists of determining the type of media and the conditions of specific media functions. Media query allows users to adjust the size to fit a specific range of output devices without changing the content itself. In other words, media query gives users a great view of the website on all kinds of displays, from smartphones to large screens. Typography is also important. Until recently, most developers used pixels to define font sizes. If the width of the website is fixed, pixels are easy to use, but responsive websites must have responsive fonts. The site font size must be related to the container width, so it can be adapted to the client screen and easily read from the mobile device.
